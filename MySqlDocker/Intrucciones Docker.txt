Hay que ejecutarlo desde el mismo directorio donde esta el archivo "docker-compose.yml" y el directorio "sql-scripts".

	
*** Conectar al contenedor MySQL ***	
docker exec -it mysqldocker mysql -u root -p1234

	docker exec -it: Ejecuta un comando dentro de un contenedor en ejecución.
	mysqldocker: Es el nombre del contenedor que hemos creado (el container_name del compose.yml).
	mysql -u root -p: Llama a MySQL con el usuario root y contraseña 1234 (el MYSQL_ROOT_PASSWORD del compose.yml).
	
	
***Levantar los contenedores con compose***	
docker-compose up -d

	-d: para ejecutar en segundo plano

	
***Para detener todos los contenedores definidos en tu archivo***	
**Esto detendrá y eliminará todos los contenedores, redes y volúmenes creados por docker-compose up**
docker-compose down	

**Si solo quieres detener los contenedores sin eliminarlos**
docker-compose stop
	Esto mantiene las bases de datos y sus registros.	
	*Si solo se ha parado se puede levantar de nuevo con docker-compose up o con docker-compose start
